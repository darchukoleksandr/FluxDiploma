About Flux

Flux is an open source desktop application intended to allow encrypted online chatting available for Windows.

It uses OpenPGP encryption to secure all communications to other users. Flux is developed by Oleksandr Darchuk and is published under the terms of the GPLv3 license. Solution features include offline user messaging, multi-user chats and social networks authentication.

Features

WebSockets

Lower overheads, facilitating real-time data transfer from and to the server.

OpenPGP

Combination of hashing, compression, symmetric-key cryptography, and public-key cryptography.

OpenID Connect

Support JSON Web Tokens authentication for multiple social networks.

Information

Secure sharing of information among people is one of the most important issues. Despite the apparent popularity and practically universal availability, computer communications are still considered insufficiently protected. In order to keep up with the general trend and maintain its position, popular messengers are starting to introduce different methods of encryption of messages one by one. As a result, each messenger has both advantages in the form of security and disadvantages in performance.

The purpose of the development is to create a software system that is designed to provide the maximum speed and security of messaging. The system is aimed at messaging for small groups of up to 50 people.
